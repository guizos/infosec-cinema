Welcome to InfoSec Cinema
InfoSec Cinema is a database of movies and information security events. Initially this tool will show you all the movies with their corresponding events. You can filter which events you would like to show by writing keywords or selecting STRIDE and ISO Control filters. You can add your own movies by contributing on the following Github repo.
We've run InfoSec cinema twice with Rogue One as our selected film. You can access the presentation used during our two sessions here.
